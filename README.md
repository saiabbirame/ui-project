# Static UI (HTML / CSS / JavaScript)

This folder contains a tiny static UI that displays a message and a picture.

Files:

- `index.html` — the page to open
- `css/style.css` — styles
- `js/main.js` — small JavaScript to change/reset the message
- `images/picture.svg` — decorative image shown on the page

How to view:

1. Open `index.html` directly in your browser (double-click or use "Open File...").

OR

2. From the `root` directory run a simple HTTP server (recommended for some browsers):

```bash
# from the project root
python3 -m http.server 8080
# then open http://localhost:8000 in your browser
```

That's it — no build step required.
